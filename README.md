# 🚦 Data-Driven Traffic Management System

An intelligent traffic management system designed to optimize traffic flow using real-time traffic density analysis and data-driven decision making. This project focuses on reducing congestion, improving signal efficiency, and enabling smarter urban traffic control.

## 📌 Overview

Traditional traffic systems use fixed timers, which often lead to:
- Traffic congestion
- Increased waiting time
- Fuel wastage
- Poor emergency handling

This project introduces a smarter approach where traffic signals can adapt dynamically based on vehicle density and traffic conditions.

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/PratikPardhi04/Data-Driven-Traffic-Management-System.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Data-Driven-Traffic-Management-System
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start the Server

```bash
npm start
```
